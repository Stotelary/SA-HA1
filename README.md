# 404 Service - Servicio no encontrado... hasta ahora.

Plataforma web para conectar profesionales con clientes que necesitan servicios online y presenciales.

## Características

- **Landing Page completa** con Hero, Servicios, Sobre Nosotros, Beneficios, Testimonios y Contacto
- **Página de detalle de servicio** con carrusel de imágenes, sistema de agendamiento y reseñas
- **Diseño responsivo** optimizado para mobile, tablet y desktop
- **Formularios funcionales** con validación y notificaciones toast
- **Navegación suave** con scroll animado
- **Paleta de colores personalizada**: #242037, #735ac0, #F95738, #F4F1DE
- **Fuente Manrope** para toda la tipografía

## Instalación

1. Descarga o clona el repositorio
2. Abre `index.html` en tu navegador
3. No requiere instalación de dependencias

## Uso

### Navegación
- El menú superior permite navegar entre las diferentes secciones
- En mobile, usa el botón de hamburguesa para abrir el menú
- Todos los enlaces tienen scroll suave

### Formulario de Contacto
- Completa el formulario en la sección de contacto
- Selecciona el tipo de consulta (cliente o proveedor)
- Al enviar, verás una notificación de confirmación

### Sistema de Agendamiento
- En la página de servicio, selecciona fecha y hora
- Elige el tipo de servicio que necesitas
- Agrega notas adicionales si es necesario
- Al agregar al carrito, verás una confirmación
